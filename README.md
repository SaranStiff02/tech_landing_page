## 📌 Project Overview

Reveldor Software Landing Page is a responsive frontend website designed for a software and training company.

The website presents company branding, service navigation, hero content, promotional media, and a training/workshop section.

---

## ✅ Features

- ✅ Responsive navigation bar
- ✅ Mobile hamburger menu
- ✅ Hero section with promotional content
- ✅ Software company landing page layout
- ✅ Training/workshop section
- ✅ Hover effects and basic interactivity
- ✅ Responsive design using CSS media queries
- ✅ Bootstrap integration
- ✅ Clean folder structure for GitHub portfolio
